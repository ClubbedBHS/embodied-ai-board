# 机器人学习中的组合推理

> 提出了一套从强结构化先验到弱先验的机器人组合推理技术体系，涵盖 Neural Task Programming、Neural Task Graphs 和 Regression Planning Networks，使机器人能够通过组合已知技能泛化到新的长时域操作任务。

## 研究问题

当前机器人学习方法大多针对单一任务进行专业化训练，缺乏跨任务泛化能力。相比之下，人类能够从日常经验中提炼可复用的抽象概念，并通过组合已知的构建模块来解决新任务。本论文的核心问题是：**如何赋予机器人系统组合推理能力，使其能够将已学技能灵活组合以解决新的复杂操作任务？**

论文将这一问题分为两个层次：(1) 利用强结构化先验（程序归纳、图模型）设计具有组合泛化能力的算法；(2) 放宽这些先验假设，使机器人能够从非结构化的视频演示和主动交互中自主学习组合结构。

## 核心方法

论文分为两部分，系统性地探讨了不同先验强度下的组合推理方法。

**第一部分：强结构化先验下的组合推理**

*Neural Task Programming (NTP)* 将任务策略建模为层次化神经程序。给定任务规范（如视频演示），NTP 递归地将规范分解为更细粒度的子任务规范，并由分层神经程序逐级执行。其核心设计是作用域约束（scoping constraint）：子任务规范必须是父任务规范的连续子序列，强制底层模块仅访问与其功能相关的信息，从而实现模块化复用。NTP 首次将 meta-learning 与神经程序归纳结合，实现了从单次演示中学习可复用层次化策略的能力。

*Neural Task Graphs (NTG)* 将操作技能表示为图中的节点，并通过技能前置条件的结构化关系实现泛化。NTG 能够从单个视频演示中推断任务图结构，并将其泛化到具有不同目标、不同物体数量的新任务场景中。

*Regression Planning Networks (RPN)* 进一步关注目标规范的分解。RPN 将符号化目标（如「将红色方块放在蓝色容器中」）通过以物体为中心的表征进行接地（grounding），并学习从初始状态到目标状态的连续规划。该方法能够实现对新任务目标的零样本泛化。

**第二部分：放松先验假设**

论文进一步探索了如何在减少结构化先验的条件下实现组合推理。首先，提出了一种从人类视频演示中同时学习复杂视觉运动技能和发现隐式组合结构的框架，实验证明该方法能够使物理机器人在厨房环境中执行长时域操作任务。其次，进一步放宽对演示数据的需求，通过主动交互表征运动基元的前置条件和后置条件，使机器人能够自主学习紧凑的规划表征。

## 主要贡献

1. 提出了 NTP 框架，首次将神经程序归纳与 meta-learning 结合，为机器人 one-shot 模仿学习提供了模块化、可复用的层次策略结构。
2. 提出了 NTG，利用图结构对技能及其前置条件建模，实现从单次视频演示到新任务的泛化。
3. 提出了 RPN，将符号化规划目标与以物体为中心的连续表征相结合，实现了对未见目标的零样本泛化。
4. 在第二部分中，展示了如何在减少结构化先验的前提下，从视频演示和主动交互中学习组合式规划表征，拓展了组合推理在更真实场景中的适用性。

## 数据与训练

论文各章节对应不同的实验设置和数据需求：

- NTP 在 PyBullet 仿真器中以程序方式生成演示轨迹，训练包含 2000 个 Block Stacking 任务实例和 256 种 Object Sorting 类别-容器映射组合。使用监督学习（行为克隆），约 100-1000 条演示轨迹。
- NTG 在类似仿真环境中验证，使用单个视频演示作为输入。
- RPN 在操作仿真任务中验证，关注零样本目标泛化。
- 第二部分中，视频演示学习框架使用人类遥控操作的演示数据。主动交互学习框架通过机器人与环境的试错交互收集数据。

所有方法均包含仿真实验，NTP 和第二部分的工作在真实机器人平台（Sawyer 臂）上进行了验证。

## 实验与结果

**仿真实验（论文各个子工作）：**

- NTP 在 Object Sorting 任务中，物体数量从 4 个增长到 40 个时保持 90%+ 成功率，Flat 基线从 40% 降至 25%。在 Block Stacking 中，NTP 在未见任务上表现（~85%）与见任务几乎持平，Flat 基线完全无法泛化（~0%）。
- NTG 从单次视频演示推断任务图，并在未见任务配置上实现了显著优于基线方法的泛化性能。
- RPN 在对新目标配置的零样本泛化实验中，成功率远超基于符号规划的基线方法。
- 视频演示学习框架在厨房仿真和真实场景中成功执行了多步骤长时域操作任务。

**真实机器人实验：**

- NTP 在 Sawyer 机器人上进行 Block Stacking（20 次试验，90% 成功率）和 Object Sorting（10 次试验，80% 成功率）实验，失败主要来自抓取和碰撞检测等操作层面，NTP 算法决策几乎无误。
- 第二部分中，物理机器人在厨房环境中成功执行了长时域操作，验证了从视频演示中学习组合技能的可行性。

## 局限性

- NTP 的底层 API 为预定义的符号化动作基元，需要人工设计，难以扩展到丰富的连续控制接口。
- 第一部分的三种方法均依赖已知的程序层次结构或图结构，无法从数据中自动发现任务分解。
- 任务规范需要人工提供（演示轨迹或视频），未集成语言指令等更自然的交互方式。
- 第二部分的方法在更少结构化先验的同时，泛化能力和稳定性可能不如第一部分中强先验引导的方法。
- 所有真实机器人实验均在受控实验室环境中完成，未在复杂日常场景中验证。

## 为什么值得关注

该博士论文为机器人学习中的**组合推理**问题提供了一份系统性的全景视图。从强结构化先验（神经程序归纳、图模型）到弱先验（从非结构化数据中自主学习），论文呈现了一条清晰的技术演进路线。NTP、NTG 和 RPN 等代表性工作在发表时均是该领域的里程碑式贡献，其核心思想——通过组合已知技能实现对新任务的泛化——至今仍是具身智能和 TAMP 研究的核心议题。对于希望理解**组合式策略学习、分层规划和机器人操作泛化**的研究者来说，这份论文是最佳的系统性入门读物之一。
