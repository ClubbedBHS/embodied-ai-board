# EgoScale：利用大规模以自我为中心的人类数据扩展灵巧操作

> 在大规模第一人称人类视频（20,854 小时）上预训练 VLA 模型，发现人类动作预测损失的 log-linear 扩展规律，结合少量对齐的人-机器人 mid-training 实现 22-DoF 灵巧手的长程操作与 one-shot 任务适应。

## 研究问题

人类行为是学习物理智能最具可扩展性的数据来源之一，但如何有效利用人类数据训练灵巧操作策略仍是一个悬而未决的问题。现有工作要么使用相对较小的人类数据集（通常几十到几百小时），要么聚焦于夹爪或低自由度手，缺乏精细的手指关节控制（Sec 1, para 2）。EgoScale 试图回答一个根本性问题：大规模人类数据是否能够作为灵巧操作的主要训练信号？如果可以，是否存在可预测的扩展规律？

## 核心方法

EgoScale 提出一个三阶段人-机器人迁移框架（Sec 2.4），核心设计包括：

**1. 统一的人体动作表征（Sec 2.1）。** 从第一人称 RGB 视频中提取相对腕部运动（SE(3)）和经重定向的灵巧手关节动作（22-DoF Sharpa 手空间）。腕部相对运动消除了绝对相机位姿的依赖，使得人类演示和机器人执行共享相同的动作抽象。手部关键点通过 CasADi + IPOPT 的优化重定向过程映射到机器人关节空间（Appendix D）。

**2. 大规模人类数据预训练 Stage I（Sec 2.2, 2.3, 2.4）。** 在 20,854 小时的第一人称人类活动视频上预训练 VLA 模型。数据来源包括大规模野外采集（覆盖 9,869 个场景、6,015 个任务、43,237 个物体）和 829 小时的 EgoDex 高精度数据集（194 个桌面任务）（Sec 2.2）。模型使用基于 flow matching 的 VLA 架构（类似 GR00T N1），以 VLM 为骨干、DiT 为动作专家，在 256 块 GB200 GPU 上训练 100K 步，batch size 8,192，lr 5e-5（Sec 2.3, 2.4）。

**3. 对齐的人-机器人 mid-training Stage II（Sec 2.2, 2.4）。** 引入一个小规模但对齐的人-机器人数据集：344 个桌面操作任务，每个任务约 30 条人类轨迹和 5 条机器人遥操作轨迹（共约 50 小时人类数据 + 4 小时机器人数据）。人类和机器人使用相同的相机配置（头盔 + 双腕相机），确保视觉观测直接可比。使用 Vive 追踪器 + Manus 手套采集人类动作。此阶段冻结 VLM 骨干，仅更新视觉编码器和 DiT 动作专家，训练 50K 步，batch size 2,048，lr 3e-5（Sec 2.4）。

**4. 任务后训练 Stage III（Sec 2.4）。** 在下游灵巧操作任务上仅用 20-100 条机器人演示进行微调，训练 10K 步，batch size 512，lr 3e-5。

## 主要贡献

1. **人类数据预训练的扩展规律（Sec 3.3, Fig 5）。** 在 1K-20K 小时的数据规模上，发现人类手部动作预测验证损失与数据量呈近乎完美的 log-linear 关系（L = 0.024 − 0.003·ln(D)，R² = 0.9983，Eq. 1），且该损失与下游真实机器人性能强相关。在当前探索范围内未见饱和迹象。

2. **有效的人-机器人迁移配方（Sec 3.2, Fig 4）。** 大规模人类预训练 + 少量对齐 mid-training 的组合，在 22-DoF Sharpa 灵巧手上平均任务完成度相比无预训练基线提升超过 55%（Abstract 中报道成功率平均提升 54%）。仅用人类预训练（无 mid-training）已超越仅用 mid-training 的基线，证明规模和多样性本身提供了强大的归纳偏置。

3. **涌现的 one-shot 迁移能力（Sec 3.4, Fig 6）。** 仅用 1 条机器人演示 + 100 条对齐的人类演示，即可在新任务上实现 88%（Fold Shirt）和 55%（Unscrewing Water Bottles）的任务完成分，而缺少预训练或 mid-training 的模型完全失败。注意此处为 one-shot 场景的任务完成分，非主实验中的 5 任务评估。

4. **跨具身泛化（Sec 3.5, Fig 7）。** 在 Unitree G1 机器人（7-DoF 三指手）上，人类预训练策略相比无预训练基线在 Pen in Bin 和 Dish Handover in Rack 两项任务上均提升超 30% 绝对成功率，表明大规模人类运动提供了可复用的、与执行体无关的动作先验。

5. **手部动作表征消融（Sec 3.6, Fig 8）。** 对比腕部-only、指尖-based 和完整关节角重定向三种表征，关节空间动作在所有任务上表现最稳定，尤其在需要精细手指关节和接触时序的任务（Tongs、Cards）上优势显著。

## 数据与训练

- **Stage I 数据：** 20,854 小时第一人称人类视频（野外采集 + 829 小时 EgoDex）。涵盖零售（20.1%）、时尚（11.8%）、维修服务（11.5%）、食品饮料（11.5%）、家居（9.5%）、建筑五金（7.7%）、食品加工（6.7%）、印刷设计（4.4%）等多个领域（Appendix C, Fig 10）。使用 SLAM + 手部姿态估计管线提取相机运动和手部轨迹（Sec 2.2）。
- **Stage II 数据：** 344 个桌面任务的对齐人-机器人数据，人类使用 Vive 追踪器 + Manus 手套，机器人通过遥操作采集（Sec 2.2）。
- **模型架构：** Flow-based VLA，VLM 骨干 + DiT 动作专家，轻量级具身特定 MLP 适配器处理本体感知和手部动作（Sec 2.3）。
- **训练配置：** Stage I：100K 步，batch size 8,192，256×GB200 GPU，lr 5e-5；Stage II：50K 步，batch size 2,048，lr 3e-5；Stage III：10K 步，batch size 512，lr 3e-5（Sec 2.4）。

## 实验与结果

- **五项核心任务评估（Sec 3.1-3.2, Fig 3-4）：** 在 Shirt Rolling（叠卷 T 恤）、Card Sorting（分牌）、Tong Fruit Transfer（夹子取水果）、Bottle Cap Unscrewing（拧瓶盖）、Syringe Liquid Transfer（注射器移液）上，Human Pretrain + Midtrain 组合平均任务完成分和成功率均显著领先。Abstract 报道平均成功率提升 54%，正文报道平均任务完成分提升超 55%。Per-task 数据见 Fig 4（需核对原图）。
- **扩展规律验证（Sec 3.3, Fig 5, Eq 1）：** 随人类预训练数据从 1K 增至 20K 小时，下游平均任务完成分从 0.30 单调上升至 0.71（Fig 5 right），验证损失呈完美对数线性下降（L = 0.024 − 0.003·ln(D)，R² = 0.9983，Eq. 1），且验证损失与机器人性能强相关。
- **One-shot 迁移（Sec 3.4, Fig 6）：** 仅 1 条机器人演示 + 100 条对齐人类演示，Fold Shirt 任务完成分达 0.88，Unscrewing Water Bottles 达 0.55（三瓶不同几何瓶身）。缺失预训练或 mid-training 的模型完全失败。
- **跨具身泛化（Sec 3.5, Fig 7）：** Unitree G1 + 7-DoF 三指手，Pen in Bin 和 Dish Handover in Rack 两项任务上人类预训练 + mid-training 显著优于仅 mid-training。注意 G1 的下半身平衡和 locomotion 由独立训练的 Homie 策略处理。
- **动作表征消融（Sec 3.6, Fig 8）：** 腕部-only 在所有任务上表现最差，指尖-based 在接触敏感任务上因映射误差导致抓取不稳定，关节空间最稳定。
- **机器人平台（Sec 2.5）：** Galaxea R1Pro 双臂轮式机器人 + 22-DoF Sharpa 灵巧手；跨具身实验使用 Unitree G1 + 7-DoF 三指手。

## 局限性

作者在结论部分（Sec 6）指出：(1) 当前探索范围未观察到饱和，但联合扩展人类数据量和模型容量可能解锁进一步增益；(2) 未探索利用无标签视频的自监督目标；(3) 具身差距仍存在，随着机器人硬件向更类人的运动学和灵巧度发展，迁移效果有望继续提升。编辑推断：当前方法依赖大规模 GPU 计算资源（256×GB200）和大规模自采人类视频数据，独立研究团队难以复现同等规模实验；代码尚未公开（标注 "Coming Soon"）。

## 为什么值得关注

EgoScale 的核心理念——**将人类视为一种可扩展的机器人学习具身**——为灵巧操作领域提供了新的范式。其发现的 log-linear 扩展规律（人类动作预测损失随数据规模对数线性下降，且与真实机器人性能高度相关，R² = 0.9983）是首个在大规模人类数据上系统验证的人-机迁移扩展律。两阶段迁移配方（大规模预训练 + 少量对齐中训练）简单有效，且涌现出 one-shot 泛化能力，暗示人类数据中的操作结构可以被有效提取并复用于多种具身。这一方向对成本高昂的灵巧操作数据采集问题提供了可扩展的替代方案。
