# FM-VLA：用于接触丰富操控的力觉记忆视觉语言动作模型

> 把力/力矩历史压缩成记忆 token，让 VLA 识别视觉上难以区分的重复接触事件。

## 研究问题

在重复按按钮、擦拭或遮挡操作中，不同历史状态可能产生几乎相同的图像。保存更多历史图像既昂贵，也无法可靠表达接触次数。论文研究能否直接利用腕部六轴力/力矩历史，为 VLA 提供非马尔可夫任务所需的事件记忆。

## 核心方法

FM-VLA 先用 VAE 对力/力矩时间序列做重建预训练，把历史压缩为固定数量的 force memory tokens，再与短期机器人状态一同投影到 VLA action expert。视觉、语言、短状态和力觉记忆共同条件化 flow-matching 动作生成，同时避免持续加入大量历史图像。

## 主要贡献

1. 把力觉历史定义为 VLA 的紧凑事件记忆，而非单步触觉输入。
2. 针对视觉歧义的重复接触任务设计真实双臂评测。
3. 通过架构和 token 容量消融分析 force memory 的作用与开销。

## 数据与训练

数据在 AgiBot G1 双臂 humanoid 上采集，机器人双腕各配备 100 Hz 六轴 wrench sensor，并使用头部和双腕三路 RGB、关节和夹爪状态。Force VAE 使用同一 demonstration dataset 的力觉序列训练。

## 实验与结果

第 4 节评估寻找被杯子遮挡的积木、重复按按钮以及按指定次数擦碗三个记忆依赖任务。摘要报告轻量力觉记忆取得超过 80% 成功率并显著优于基线；正文同时比较视觉记忆、短状态、单 token/多 token 设计和推理开销。

## 局限性

1. 仅覆盖三个专门设计的任务和单一机器人平台。
2. 固定 8-token bottleneck 未验证数百次接触事件的超长任务。
3. Force VAE 只在当前 demonstration 数据上训练，跨机器人和传感器泛化未知。

## 为什么值得关注

这篇论文把一个容易被忽略的问题说得很清楚：视觉记忆不等于物理交互记忆。力觉历史可能是接触丰富 VLA 中成本更低、语义更直接的状态通道。
